## 📝 Cara Upload (Tanpa Git):

### 1. Buka GitHub.com
   - Login
   - Buat repository baru: `furniture-price-generator`
   - Public
   - Klik "Create repository"

### 2. Upload Files
   - Klik "uploading an existing file"
   - **Drag & drop 6 file ini**:
     1. `streamlit_app.py`
     2. `material_descriptions.py`
     3. `requirements-streamlit.txt`
     4. `README.md`
     5. `README_STREAMLIT.md`
     6. `QUICKSTART_STREAMLIT.md`
   
   - Commit message: `Initial commit: Furniture Price Generator`
   - Klik "Commit changes"

### 3. Deploy ke Streamlit Cloud
   - Buka: https://streamlit.io/cloud
   - Sign in with GitHub
   - New app → Pilih repository
   - Main file: `streamlit_app.py`
   - Deploy!
